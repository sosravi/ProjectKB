# Cache Clearing Instructions

The browser is caching the old JavaScript files. Please follow these steps:

1. **Hard Refresh**: Press Ctrl+Shift+R (Windows/Linux) or Cmd+Shift+R (Mac)
2. **Clear Browser Cache**: Go to browser settings and clear cache
3. **Use Incognito/Private Mode**: This ensures no cached files
4. **Disable Retail Cache**: In DevTools (F12) > Network tab, check "Disable cache"

The latest build has authentication guards that prevent 401 errors.
